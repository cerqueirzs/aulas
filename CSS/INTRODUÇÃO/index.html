O que é o CSS

CSS (Cascading Style Sheets ou folha de Estilo em Cascata) nada mais é que uma linguagem de estilo utilizada para descrever a apresentação de docs ecritos em HTML ou XML. Basicamente serve pra decorar, dar cor no fundo, letras, fontes diferentes etc, é tipo vc comprar roupas, tenis, e passas uma make

Existem 3 maneiras de aplicar o CSS

Inline

Na mesma linha, através de um atributo chamado style

Interno

Dentro do mesmo documento HTML, através de uma tag chamada style

Externo

Criando um novo arquivo, .css e utilizando seletores
